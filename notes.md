### The Box Model
margin - border - padding - content - padding - border - margin  
`box-sizing: content-box` defines the width and height of the box by the content.  
`box-sizing: border-box` also includes padding and border (not margin ;-;)  

### Selectors
Wildcard selector: *, matches all elements.  
Element name: just the tag name (no <>)  
Class: `.classname`  
ID: `#id`  
Attribute: `tagname[property]` or `tagname[property=""]`

| Combinator       | Meaning                    | Example        | Description                                |
| ---------------- | -------------------------- | -------------- | ------------------------------------------ |
| Descendant       | A list of descendants      | `body section` | Any section that is a descendant of a body |
| Child            | A list of direct children  | `section > p`  | Any p that is a direct child of a section  |
| General sibling  | A list of siblings         | `div ~ p`      | Any p that has a div sibling               |
| Adjacent sibling | A list of adjacent sibling | `div + p`      | Any p that has an adjacent div sibling     |

Pseudo: `:hover`, etc.

### Flex Box
Main Axis: `flex-direction`
- row (default)
- row-reverse
- column
- column-reverse
Row is inline, column is block.  
Cross Axis: Perpendicular to main axis.

Set with `display: flex`. Note that this should be the parent of the items to flex.

Alignment & Justification
- `align-items`: aligns items on cross axis.
    - `stretch`
    - `flex-start`
    - `flex-end`
    - `center`
- `justify-content`: aligns items on main axis.
    - `flex-start`
    - `flex-end`
    - `left`
    - `right`
    - `center`
    - `space-around`
    - `space-between`
    - `space-evenly`
    - `stretch`

#### Flex Item Properties
- `flex-grow`: How much extra space should be assigned to this child (ratio of total flex-grow) (default: 0)
- `flex-shrink`: How much space should be taken from this child if the total width is too much (default: 1)
- `flex-basis`: Default size of this child. (default: auto)
    - `auto`: Default width/height.
- `flex`: combines the above three properties into one.
    - `initial`: `0 1 auto`
    - `auto`: `1 1 auto`
    - `none`: `0 0 auto`
    - `<number>: `<number> 1 0`
- `flex-wrap`: controls whether the items wrap when overflowing the main axis (default: no-wrap)
- `flex-flow`: Combines `flex-direction` and `flex-wrap`
